## 플랫폼별 유의사항

### Instagram Feed
- 이미지 비율: 1:1 또는 4:5 권장
- 캡션 첫 줄이 가장 중요 — "더 보기" 전에 보이는 125자
- 해시태그: 캡션 마지막 또는 첫 댓글에 삽입
- 추천 해시태그: #식단관리 #밀그램 #닥터스핏 #다이어트앱 #스포츠영양 #식단기록 #헬스식단 #AI식단분석

### Threads
- 캡션 위주, 이미지는 보조
- 줄바꿈 활용해 읽기 쉽게 구성
- 해시태그 최소화 (1~2개 또는 생략)
- 크레딧 문구("운동하는 의사가 만든") 앞쪽에 배치 — Threads는 텍스트 신뢰도가 높은 플랫폼

### Facebook Feed
- 본문 캡션 길어도 괜찮음 (Facebook은 긴 글 소화 가능)
- Headline + Description 모두 노출되므로 구체적으로 작성
- 나이 타깃 30~45세 기준 → 신뢰 강조 문구 더 효과적

## 이미지 구성 가이드

앱 사용 사진 선택 기준:
- 실제 사용 화면 — 과도한 보정 금지
- 추천 화면: 메뉴 입력 + 분석 결과 나란히 / 챗봇 대화 화면 / 주간 리포트 화면
- 텍스트 오버레이 사용 시 핵심 문구 1줄만 ("외식도 기록됩니다" / "닥터스핏 AI 챗봇 내장")
- 배경: 음식 사진 + 앱 화면 합성도 가능

## 주의 사항

- "살이 빠집니다", "2주 만에 효과" 같은 다이어트 결과 단정 표현 금지 (Meta 정책 위반)
- "의료 앱", "처방", "치료" 표현 금지
- "운동하는 의사 + 스포츠 영양 코치가 만든 식단 관리 앱" 표현은 문제 없음
