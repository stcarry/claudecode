# 썸네일 컨셉 — 쉽게 운동해야 체력이 좋아집니다

**연결 대본:** `2026-03-23 롱폼_체력훈련_크로스핏하이록스.md`
**영상 제목:** 쉽게 운동해야 체력이 좋아집니다
**타겟:** 크로스핏/하이록스 입문~중급자
**카테고리:** 교육/역설적 주장
**핵심 전략:** 반직관 + 호기심 갭

## Concept 1: 의심 유발 — "진짜로?"

**Visual Hook:**
본인 얼굴이 "이거 진심이야?" 표정으로 카메라 정면 응시. 옆에 심박수 Zone 2 수치(60~70%) 그래픽 삽입. 배경은 여유롭게 로잉하는 장면 (고강도 아닌 느긋한 느낌).

**Text Overlay:**
```
진짜입니다
```
- Font: Bebas Neue
- Position: 우측 상단
- Color: 형광 노랑 `#FFD700` on 다크 배경

**Facial Expression:**
한쪽 눈썹 올리고 살짝 씩 웃는 "나만 알고 있던 비밀" 표정. 자신감 있는 아이컨택.

**Color Scheme:**
- Primary: 다크 네이비 `#0D1B2A`
- Accent: 형광 노랑 `#FFD700`
- Background: 어두운 짐 or 로잉 장면 (채도 낮춤)

**Composition:**
```
+------------------+
|         | 진짜   |
| [FACE]  | 입니다 |
|         | ❤️65% |
+------------------+
```

**Why It Works:**
제목의 역설적 주장("쉽게 해야 좋아진다")을 "진짜입니다"로 받아치면 제목+썸네일이 하나의 문장처럼 읽힘. 시청자가 "진짜? 왜?"로 클릭 유도.

**Potential CTR:** 6~8%

---

## Concept 2: 비교 충격 — 열심히 vs 쉽게 ⭐ 1순위 추천

**Visual Hook:**
좌우 분할. 왼쪽: 땀 흘리며 지쳐 쓰러지는 장면 + ❌ 표시 (붉은 톤). 오른쪽: 여유롭게 페달 밟거나 걷듯 달리는 장면 + ✅ 표시 (초록/파란 톤).

**Text Overlay:**
```
이게 맞습니다
```
- Font: Montserrat Bold
- Position: 하단 중앙
- Color: 흰색 on 검정 배경 블록

**Facial Expression:**
중앙 하단에 작게 본인 얼굴 — 오른쪽(✅)을 가리키는 제스처.

**Color Scheme:**
- Primary Left: 레드 `#C0392B`
- Primary Right: 틸 `#1ABC9C`
- Text Block: 블랙 `#111111`

**Composition:**
```
+------------------+
| ❌지침  | ✅여유 |
| [힘듦]  | [쉬움] |
|   이게 맞습니다  |
+------------------+
```

**Why It Works:**
Before/After 구도인데 반전이 있음 — 보통 "힘든 게 After(좋은 것)"인데 여기선 "쉬운 게 ✅". 이 역전 구도가 시각적 충격으로 작동. 제목과 완벽하게 연결. 모바일에서도 ❌/✅ 분할이 한눈에 읽힘.

**Potential CTR:** 7~9%

---

## Concept 3: 숫자 폭탄 — 80%

**Visual Hook:**
굵은 "80%"가 화면 절반 이상을 차지. 본인 얼굴이 오른쪽 1/3에서 80%를 손가락으로 가리키는 포즈. 아래에 작은 글씨로 "저강도로만" 텍스트.

**Text Overlay:**
```
80%
저강도로만
```
- Font: Bebas Neue (숫자 극대화)
- Position: 좌측 중앙 (숫자), 좌측 하단 (보조텍스트)
- Color: 오렌지 `#FF6B00` on 다크 배경

**Facial Expression:**
숫자를 가리키며 놀란 듯한 "이 숫자가 핵심이야" 표정. 눈 살짝 크게.

**Color Scheme:**
- Primary: 블랙 `#111111`
- Accent: 오렌지 `#FF6B00`
- Background: 어두운 단색 or 블러 처리된 짐

**Composition:**
```
+------------------+
|  80%    |        |
|저강도로만| [FACE]|
|         |  👆   |
+------------------+
```

**Why It Works:**
제목 "쉽게 해야 좋아진다"의 근거를 숫자로 선제 제시. "80%가 뭔데?" 호기심 갭 생성. 모바일에서도 숫자가 압도적으로 크게 읽힘.

**Potential CTR:** 6~8%
