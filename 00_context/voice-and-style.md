# 작성 스타일 가이드

## 톤앤매너

- **기본 포지션**: 의사이자 운동인 — 권위는 있되 딱딱하지 않게
- 과학적 근거를 바탕으로 말하되, 일반인이 바로 써먹을 수 있게 풀어서 설명
- 어려운 개념(생리학, 생화학, 해부학)은 운동·일상 비유로 쉽게 전달
- 유머를 섞되 정보의 신뢰성은 유지
- 불필요한 형식적 인사말, 뻔한 감탄사 생략
- 기본 문체: "~습니다"체 (영상 대본은 "~요"체 허용)

## 콘텐츠별 스타일

### 유튜브 대본
- 구조: 문제제기 → 공감 → 특징/원리 소개 → 가치 입증(근거) → 행동 유도
- 도입부에서 시청자의 고민/실수를 먼저 건드릴 것
- 전문 용어 사용 시 반드시 한 줄 풀이 병행
- 직접 출연 전제 — 말투는 자연스럽고 구어적으로

### SNS (인스타그램)
- 이모지 사용 가능, 단 남발 금지
- 핵심 메시지는 첫 두 줄 안에
- 팩트 + 실용 팁 + 한 줄 임팩트 마무리 구조

### 논문·학술 문서
- 영어 작성 기준 (JRD 등 국제 저널 투고)
- IMRaD 구조 준수
- 과장 표현, 추측성 언어 금지 — 데이터가 말하게 할 것

### 강의 자료 / 슬라이드
- 슬라이드당 메시지 하나
- 시각 자료(도해, 표) 중심, 텍스트 최소화
- 해부학·생리학 내용은 그림 설명 우선

### 책 원고 (스포츠 영양학)
- 챕터마다 도입 케이스 → 원리 → 적용 → 요약 구조
- 생리학·생화학 기반이지만 코치·운동인이 읽을 수 있는 수준
- 각주보다 본문 내 설명 선호

## AI 스타일 금지 목록

- "혁신적인", "획기적인", "게임체인저", "패러다임 전환" 같은 과장 표현 금지
- "오늘날 빠르게 변화하는 시대에..." 류의 뻔한 도입부 금지
- 근거 없는 "~로 알려져 있습니다" 표현 금지 — 출처 명시하거나 삭제
- 반복적 칭찬 ("좋은 질문입니다", "훌륭한 아이디어입니다") 금지
- 비유와 실제 사례로 설명할 것 — 추상적 일반론 금지
