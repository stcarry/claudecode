# 스킬: Meta 광고 세팅 (MealGram)

이 스킬은 MealGram 앱 홍보를 위한 Meta(인스타그램/페이스북) 광고를 세팅할 때 따르는 방법론입니다.

---

## 1. 캠페인 목표 선택 기준

| 목표 | 선택 기준 | MealGram 적용 시점 |
| --- | --- | --- |
| 앱 설치 (App Installs) | 신규 설치 유도가 1순위일 때 | 기본값 — 항상 우선 고려 |
| 트래픽 (Traffic) | 링크인바이오 또는 앱스토어 페이지 유입 | 앱 설치 캠페인 성과 낮을 때 보조 |
| 인지도 (Awareness) | 브랜드 노출이 목적, 전환 측정 불필요 | 신규 런칭/업데이트 발표 시 |
| 전환 (Conversions) | 구독 결제 유도 목표 | 무료 사용자 → 유료 전환 캠페인 |

---

## 2. 타깃 오디언스 설정

### 기본 타깃 (Cold Audience)

- **나이**: 20~45세
- **성별**: 전체 (초기에는 전체로 시작 후 성과 기반 조정)
- **지역**: 대한민국
- **언어**: 한국어

**관심사 타깃 (조합하여 사용)**:
- 다이어트, 체중 감량, 칼로리 계산
- 헬스, 피트니스, 근력 운동
- 영양학, 건강식, 건강한 식습관
- 케틀벨, 크로스핏, 홈트레이닝
- 건강 앱, 피트니스 앱

**행동 타깃**:
- 모바일 앱 활성 사용자
- 건강/피트니스 앱 구매 이력

### 리타깃 오디언스 (Warm Audience)

| 오디언스 | 소스 | 활용 목적 |
| --- | --- | --- |
| 인스타그램 프로필 방문자 (30일) | 닥터스핏 인스타 | 앱 설치 유도 |
| 영상 시청자 (25% 이상, 30일) | 릴스/광고 영상 | 설치 전환 |
| 앱 설치 유사 오디언스 (LAL 2~5%) | 기존 설치자 | 신규 유사 타깃 확장 |
| 유료 미전환 사용자 | 앱 이벤트 | 유료 구독 유도 |

---

## 3. 광고 소재 유형별 세팅

### 영상 (릴스형)

- **비율**: 9:16 세로형
- **길이**: 15~40초 (30초 이내 권장)
- **자막**: 반드시 삽입 (무음 시청 대비)
- **첫 3초**: 텍스트 또는 강렬한 장면으로 스크롤 정지 유도
- **파일 형식**: MP4, 최소 1080x1920

### 정지 이미지 (피드/스토리)

- **비율**: 1:1 (피드) / 9:16 (스토리)
- **텍스트 비율**: 화면의 20% 이하 권장
- **해상도**: 최소 1080x1080 (피드 기준)

### 캐러셀

- **슬라이드 수**: 3~5장 권장
- **첫 장**: 가장 강력한 메시지 (클릭 유도)
- **마지막 장**: CTA + 앱스토어 아이콘/가격 안내

---

## 4. 광고 카피 구조

Meta 광고 카피는 3단 구조로 작성한다.

**1) 헤드라인 (첫 줄 — "더 보기" 전에 보이는 부분)**
- 타깃의 문제 또는 궁금증으로 시작
- 20자 이내 권장
- 예시: "외식해도 식단 기록되나요? → 됩니다"
- 예시: "닥터스핏한테 식단 물어볼 수 있다면?"

**2) 본문**
- 기능 1~2개 + 가격 언급
- 3~5줄 이내
- 예시: "MealGram은 메뉴 이름 그대로 입력하면 AI가 분석해드립니다. 닥터스핏 페르소나 AI 챗봇도 내장. 월 8,900원으로 무제한 이용."

**3) CTA 버튼**
- 앱 설치 캠페인: "지금 설치" 또는 "앱 다운로드"
- 트래픽 캠페인: "자세히 알아보기"
- 전환 캠페인: "구독 시작"

---

## 5. 예산 및 입찰 전략

### 초기 테스트 단계

- 일예산: 1만~3만원/일 (소재당)
- 소재 2~3개 동시 테스트 후 성과 기반으로 집중
- 기간: 최소 7일 이상 운영 후 판단 (최적화 학습 기간 필요)

### 입찰 방식

- 초기: **최저 비용 (Lowest Cost)** — Meta 자동 최적화
- 안정화 후: **비용 한도 (Cost Cap)** — 목표 CPI 설정 후 관리

### 성과 판단 기준

| 지표 | 기준 | 액션 |
| --- | --- | --- |
| CTR (클릭률) | 1% 이하 | 소재/카피 교체 |
| CPI (설치당 비용) | 목표치 초과 3일 지속 | 타깃 또는 소재 변경 |
| 빈도 (Frequency) | 3 초과 | 소재 교체 또는 타깃 확장 |
| ROAS | 유료 전환 캠페인 기준 설정 | 예산 증액/감액 판단 |

## 7. 세팅 시 주의 사항

- "의사가 만든" 표현은 Meta 광고 심사에서 의료 광고로 분류될 수 있어 주의
- 체중 감량·질병 치료 효과 단정 표현은 Meta 정책 위반 가능 → 조건부 표현 사용
- 앱 스토어 연동 이벤트 (설치, 구독 등) Meta 픽셀 또는 SDK 연동 여부 사전 확인
- 광고 승인 후 첫 24~48시간은 학습 단계 — 성과 판단 보류
