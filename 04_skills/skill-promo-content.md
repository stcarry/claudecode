# 스킬: 홍보 콘텐츠 기획 (MealGram)

이 스킬은 MealGram 앱 홍보용 콘텐츠를 기획할 때 따르는 방법론입니다.

---

## 1. 기획 전 필수 확인 사항

- 이번 콘텐츠의 목표: 앱 설치 유도 / 인지도 확산 / 재방문 유도 중 무엇인가?
- 집행 플랫폼: 인스타그램 릴스 / 유튜브 쇼츠 / Meta 광고 (피드/릴스) 중 무엇인가?
- 소재 유형: 영상(숏폼) / 정지 이미지(앱 사용 화면) / 캐러셀 중 무엇인가?

---

## 2. 콘텐츠 각도 (Angle) 선택

MealGram의 핵심 차별점 3가지. 기획 시 하나의 각도에만 집중한다.

| 각도 | 핵심 메시지 | 적합한 타깃 |
| --- | --- | --- |
| 외식 메뉴 인식 | "메뉴 이름 그대로 입력, AI가 분석" | 외식이 잦은 직장인, 다이어트 시도자 |
| AI 챗봇 (닥터스핏 페르소나) | "의사한테 직접 식단 물어보는 것처럼" | 닥터스핏 팬, 정확한 정보 원하는 사람 |
| 앱 가벼움 + 기록 편의성 | "복잡한 기능 없이, 기록만 계속" | 앱 이탈 경험자, 다이어트 지속 어려운 사람 |

---

## 3. 소재 유형별 기획 방법

### 영상 (릴스 / 쇼츠)

script-strategy-shortform.md의 홍보형 구조 사용:
1. 문제제기 (0~5초) — 앱 소개 없이 타깃 불편함으로 시작
2. 공감 (5~12초) — 감정선 연결
3. 특징 소개 (12~28초) — 기능 1~2개, 화면 녹화 병행
4. 가치 입증 (28~33초) — 의학 기반 설계, 가격 언급
5. 후기 + CTA (33~40초) — 구체적 후기 + 링크인바이오/앱스토어

화면 녹화 포인트:
- 메뉴 이름 입력 → 분석 결과 화면
- AI 챗봇 대화 화면
- 주간 리포트 화면 (유료 기능 티저)

### 정지 이미지 (앱 사용 화면 스크린샷)

구성 원칙:
- 실제 사용 화면 그대로 사용 (과도한 보정 금지)
- 텍스트 오버레이: 핵심 메시지 1줄 + 부연 1줄 이내
- 배경: 앱 화면 단독 or 음식 사진 + 앱 화면 합성

추천 조합:
- 외식 음식 사진 + 입력/분석 결과 화면 나란히
- 챗봇 대화 캡처 + "닥터스핏한테 직접 물어보세요" 텍스트

### 캐러셀 (슬라이드형)

구조 (5~7장):
1. 표지: 훅 문구 ("식단 기록, 이렇게 하면 포기 안 합니다")
2~4. 기능 소개 각 1장씩 (기능 + 해결 문제 1:1)
5. 가격 안내 (무료/유료 구분 명확히)
6. 후기 1개
7. CTA ("지금 앱스토어에서 MealGram 검색")

---

## 4. 카피 작성 원칙

- 첫 줄은 타깃의 문제나 감정에서 시작 — 앱 이름/기능 소개로 시작 금지
- "혁신적인", "완벽한", "최고의" 같은 과장 표현 금지
- 가격은 구체적으로 ("저렴하게" 대신 "월 8,900원")
- 의학적 효과 단정 금지 ("살이 빠집니다" → "식단 관리가 수월해집니다")
- CTA는 하나만: 링크인바이오 또는 앱스토어 검색 중 택 1

## 6. 기획 시 금지 사항

- 환자 경험·치료 결과를 후기로 사용하는 것
- "의사가 만든 앱" 강조 시 의료법상 의료 기기 광고로 오해받을 수 있는 표현
- 구체적 다이어트 효과 수치 단정 ("2주에 2kg 감량")
- 유료 기능을 무료인 것처럼 오해하게 만드는 구성
