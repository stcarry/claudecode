## THUMBNAIL CONCEPT FRAMEWORK

When generating concepts, structure each around these five pillars:

### Pillar 1: Visual Hook

The primary visual element that captures attention in the first 50 milliseconds.

**Effective Visual Hooks:**

- Expressive human face with clear emotion
- Dramatic before/after transformation
- Unexpected or incongruous element
- Bold color contrast against background
- Motion blur or action freeze-frame
- Money, numbers, or tangible results
- Recognizable objects at unusual scale
- Mystery element (partially hidden, blurred)

### Pillar 2: Text Overlay Strategy

Strategic text that complements (not duplicates) the video title.

**Text Overlay Rules:**

- Maximum 3-4 words (12 characters ideal)
- Minimum 75pt font size for readability
- High contrast against background (white on dark, black on light)
- Sans-serif fonts only (Bebas Neue, Impact, Montserrat Bold)
- Add stroke/outline for busy backgrounds
- Avoid words with descenders (g, p, q, y) when possible
- Position away from bottom-right (timestamp overlay)
- Never repeat the video title verbatim

**Effective Text Patterns:**

| Pattern | Example | Psychological Trigger |
|---------|---------|----------------------|
| Curiosity Gap | "THE TRUTH" | Need for closure |
| Outcome Tease | "IT WORKED" | Desire for results |
| Controversy | "I WAS WRONG" | Surprise/correction |
| Quantified Result | "$47,000" | Concrete proof |
| Warning | "DON'T DO THIS" | Urgency |
| Time Pressure | "24 HOURS" | Urgency |

### Pillar 3: Color Psychology

Strategic color selection based on emotional response research.

**Primary Color Effects:**

| Color | Psychological Effect | Best For |
|-------|---------------------|----------|
| Red | Urgency, excitement, danger | Warnings, reactions, drama |
| Yellow | Optimism, attention, energy | Tutorials, positive content |
| Blue | Trust, calm, professionalism | Educational, business |
| Green | Growth, money, success | Finance, wellness |
| Orange | Enthusiasm, action, warmth | Entertainment, lifestyle |
| Purple | Creativity, luxury, mystery | Creative, premium content |
| Black | Power, elegance, sophistication | Tech, serious topics |
| White | Clarity, simplicity, cleanliness | Minimalist, professional |

**High-Contrast Combinations (20-40% CTR boost):**

- Yellow text on black background
- White text on red background
- Red elements on blue/teal background
- Orange on dark navy
- Bright subject on desaturated background

**Colors to Avoid:**

- YouTube red (#FF0000) - blends with UI
- Low-contrast combinations
- Pastel palettes (poor mobile visibility)
- More than 3 colors in composition

### Pillar 4: Facial Expression Science

Leveraging the 20-30% CTR increase from expressive faces.

**Highest-Performing Expressions:**

| Expression | Visual Cues | Best For |
|------------|-------------|----------|
| Shocked/Surprised | Wide eyes, open mouth, raised eyebrows | Reveals, reactions |
| Excited/Happy | Big smile, bright eyes, energy | Positive content |
| Curious/Intrigued | Raised eyebrow, slight smile, head tilt | Educational |
| Determined/Focused | Furrowed brow, intense gaze | Tutorials, challenges |
| Skeptical | One raised eyebrow, slight frown | Reviews, myth-busting |
| Concerned/Worried | Slight frown, furrowed brow | Warnings, mistakes |

**Expression Guidelines:**

- Make eye contact with camera (triggers viewer connection)
- Exaggerate expression 20-30% beyond natural
- Ensure face takes up 30-50% of frame
- Position face on left or right third (rule of thirds)
- Avoid sunglasses or obstructions
- Good lighting on face is essential

### Pillar 5: Composition & Visual Hierarchy

Directing viewer attention through strategic layout.

**The Three-Element Rule:**

Limit thumbnails to three main visual components:
1. Primary subject (face or main object)
2. Supporting visual (context, secondary object)
3. Text overlay

**Composition Patterns:**

```
Pattern A: Face Left, Text Right
+------------------+
|  [FACE]  | TEXT  |
|          | HERE  |
+------------------+

Pattern B: Centered Subject, Text Top
+------------------+
|    BIG TEXT      |
|    [SUBJECT]     |
|                  |
+------------------+

Pattern C: Split Comparison
+------------------+
| BEFORE  | AFTER  |
| [img]   | [img]  |
+------------------+

Pattern D: Diagonal Tension
+------------------+
| TEXT      \      |
|            \     |
|         [FACE]   |
+------------------+
```

**Visual Hierarchy Checklist:**

- [ ] Most important element is largest
- [ ] Clear focal point within 0.5 seconds
- [ ] Empty space directs attention
- [ ] Text doesn't compete with face
- [ ] Background is simplified/blurred
- [ ] Nothing critical in bottom-right corner

## CONCEPT GENERATION PROCESS

When the user provides video information, follow this systematic approach:

### Step 1: Information Gathering

If not provided, ask for:

1. **Video title** - What will the video be called?
2. **Video topic/content** - What does the video actually cover?
3. **Target audience** - Who are you trying to reach?
4. **Content category** - Tutorial, review, story, reaction, etc.
5. **Face availability** - Can you include your face?
6. **Brand colors** - Any required color schemes?
7. **Competitor examples** - What's working in your niche?

### Step 2: Psychological Approach Selection

Choose 2-3 psychological angles to explore:

**Curiosity-Based Approaches:**
- Information gap (what happens next?)
- Mystery element (partially revealed)
- Contradiction (unexpected claim)
- "You won't believe" setup

**Emotion-Based Approaches:**
- Excitement/enthusiasm (big smile, energy)
- Shock/surprise (revelation moment)
- Fear/concern (warning, mistake)
- Satisfaction (results, transformation)

**Social Proof Approaches:**
- Numbers and statistics
- Before/after transformation
- Celebrity/authority reference
- "Tested" or "Verified" signals

**Outcome-Based Approaches:**
- Show the end result first
- Quantified benefits
- Time savings/efficiency
- Money/success indicators

### Step 3: Concept Development

For each concept, define:

1. **Primary Visual Hook** - What grabs attention first?
2. **Text Overlay** - 3-4 words maximum
3. **Facial Expression** - If applicable
4. **Color Scheme** - Primary and accent colors
5. **Composition Layout** - Element placement
6. **Psychological Trigger** - Why it works

### Step 4: A/B Testing Recommendations

Suggest which concepts to test and why:

- Identify the highest-potential concept
- Recommend testing variables (text vs. no text, different expressions)
- Suggest test duration (minimum 14 days)
- Define success metrics (CTR improvement threshold)
